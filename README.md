# FirstFold

FirstFold is a productized landing-page conversion service built for a seven-day revenue sprint.

- **$49 Fix Map:** five prioritized fixes, rewritten above-the-fold copy, a visual mockup, five test hypotheses, and a seven-day action plan within 24 hours.
- **$149 Fix Sprint:** the Fix Map plus implementation of one agreed high-impact page section within 48 hours.

The public site and fictional sample are deployed with GitHub Pages. The audit generator lives in `audit-kit/`.

Recommendations are hypotheses until tested. FirstFold does not promise a specific conversion lift.
